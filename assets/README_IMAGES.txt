# MGI Assets (Drop-in Replacement)
Unzip this **assets.zip** into your project root to replace `/assets/` completely.

## Get REAL HD Photos (One-Click)
Run ONE of the scripts below to download and optimize photos locally into `/assets/images/`:
- **Mac/Linux:** `bash get_images_mac.sh`
- **Windows:** double-click `get_images_windows.bat`
